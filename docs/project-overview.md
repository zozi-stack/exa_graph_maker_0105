## Purpose

Evals Plotter is an internal tool designed for creating publication-quality evaluation charts. It allows users to:

1. **Create multiple charts** on a canvas workspace
2. **Edit chart data** in real-time via a control panel
3. **Drag and position** charts freely on the canvas
4. **Export charts** as high-resolution PNG images
5. **Upload custom logos** for companies/data points
